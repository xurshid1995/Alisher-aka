🖥️ Hosting Widget Demo

Bu sahifa widget ishlashini test qilish uchun yaratilgan

⚠️ MUHIM:

Widgetning ko'rinishi uchun mijozning status_token aktiv bo'lishi va server ishlab turishi kerak!

📋 Widget nima qiladi?

  • Mijozning hozirgi balansini real-time ko'rsatadi
  • Server holatini monitoring qiladi (Faol/O'chiq/Bloklangan)
  • Qolgan kunlar sonini hisoblaydi
  • Oylik to'lov miqdorini ko'rsatadi
  • Telegram bot orqali to'lov qilish imkonini beradi
  • Support bilan bog'lanish uchun QR code
  • 3 tilda: O'zbek lotin, O'zbek kirill, Rus
  • Mobil qurilmalarda drag & drop qilish mumkin

🎨 Widget ranglari (avtomatik):

  • 🟢 Yashil - 7+ kun qolgan (Balans yaxshi)
  • 🟡 Sariq - 3-7 kun qolgan (Ogohlantirish)
  • 🔴 Qizil - 0-3 kun qolgan (Xavfli)
  • ⛔ To'q qizil - Balans tugagan

💻 Widget kodini qanday olish kerak?

Admin panel orqali:

  1. Brauzerda /hosting sahifasiga kiring
  2. Kerakli mijozni ro'yxatdan tanlang
  3. "✏️ Tahrirlash" tugmasini bosing
  4. Modal oynada "Widget kodi" bo'limi paydo bo'ladi
  5. "📋 Nusxalash" tugmasi bilan kodini nusxalang

🚀 Widget o'rnatish bo'yicha qo'llanma

  1. HTML saytiga qo'shish: </body> tegidan oldin widget kodini joylashtiring
  2. WordPress saytiga: Theme Editor → footer.php faylida </body> dan oldin qo'shing
  3. React/Vue/Angular: index.html faylida yoki dinamik ravishda script yuklang
  4. Pozitsiya: data-position="bottom-right" (bottom-left, top-right, top-left)
  5. Til: data-lang="uz_latin" (uz_cyrillic, ru)

📝 Kod misoli:

<!-- O'ng pastda (default) --> <script src="https://your-domain.com/static/js/hosting-widget.js" data-token="MIJOZ_TOKENI"></script> <!-- Chap pastda, rus tilida --> <script src="https://your-domain.com/static/js/hosting-widget.js" data-token="MIJOZ_TOKENI" data-position="bottom-left" data-lang="ru"></script>

🔧 Texnik tafsilotlar:

Backend API: GET /api/hosting/widget/<token>

Update interval: Har 5 daqiqada avtomatik yangilanadi

CORS: Access-Control-Allow-Origin: * (Har qanday domendan ishlaydi)

Cache: Brauzer localStorage'da pozitsiya saqlanadi (mobil)

Browser support: IE11+, Chrome, Firefox, Safari, Edge
